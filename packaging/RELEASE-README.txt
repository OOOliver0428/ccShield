ccShield v2.1.0 Windows x64 便携版
=================================

1. 双击 ccShield.exe。
2. 程序会自动打开浏览器；首次使用请在页面扫码登录。
3. 运行窗口中输入 O 可重新打开页面，输入 D 可打开数据目录，输入 Q 可安全退出。

用户数据
--------

Cookie、快捷房间和日志保存在：
%LOCALAPPDATA%\ccShield

删除整个程序目录不会删除个人数据。需要彻底清理时，请先退出 ccShield，
再手动删除上述用户数据目录。

安全提示
--------

- ccShield 仅监听 127.0.0.1，不应开放到局域网或公网。
- 本版本未提供 Windows 代码签名，首次运行可能出现 SmartScreen 提示。
- 请只从 https://github.com/OOOliver0428/ccShield/releases 下载，并核对 SHA-256。
- 本工具是非官方第三方工具，与哔哩哔哩无隶属或授权关系。
