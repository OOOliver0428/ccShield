# 更新日志

本项目遵循 `主版本.次版本.修订版本` 的版本格式。

## v2.0.2 — 2026-07-13

### 新增

- 新增快捷房间配置面板，支持输入短号或正常房间号。
- 新增房间号一键验证，可在不连接直播间的情况下查看主播、真实房间号、直播标题和开播状态。
- 新增快捷房间卡片，可从未连接状态直接进入常用直播间。
- 新增“一键添加当前房间”，将正在管理的直播间保存为快捷入口。
- 快捷房间信息持久化到本机 `config/quick_rooms.json`，并提供配置示例文件。

### 改进

- 新版金毛骑士产品图标替换原有文字占位标识，并同步用于登录页、工作台顶栏和浏览器标签页。
- 一键启动器会在 5173 被占用时自动尝试 5174、5175 等后续端口，并打开实际地址。
- 完善电脑版竖屏布局，适配旋转显示器和 Windows 缩放场景。
- 将“弹幕与房管名单正在实时同步”修正为“弹幕与禁言名单正在实时同步”。
- 修复深色模式下快捷房间验证结果中的真实房间号和直播标题对比度不足的问题。

### 安全与约束

- 快捷房间测试只调用房间信息解析接口，不会连接直播间或执行任何房管操作。
- 快捷房间初版不提供删除 API 或页面删除按钮；删除记录需要停止程序后手动编辑配置文件。

## v2.0.1 — 2026-07-13

## v2.0.0 — 2026-07-13
